기획 계획서 # 001

1. 홈페이지 : 네이버블로그 사용 : 새로 만들던가 알아서 하셈.

	홈페이지 기능은 테트리스(작명 안함)설치파일 다운로드, 공지사항, 피드백박는 용도 등 으로 사용될 예정임.
	홈페이지 만드는 것은 추후에 많이 변경할 예정 : 이유는 정보부족임.

2. 설치파일 만들기 : 이것은 추후에 변경할 예정 : 이유 정보부족

	설치파일은 테트리스 실행파일과, 실행을위한 파일, 서버로의 연결 등등을 포함함.
	install 과 uninstall 가능 + 제어판에 응용프로그램으로써 추가 아무튼 잘모르겟음 정보가 없음.

3.  서버 : C++로 구현하거나 알아서 합시다. 리눅스로 하던가.

4. 클라이언트 : 로그인 및 서버와의 통신 + 게임실행이 가능함. java script로 구현 또는 C++, 기획자는 java추천함.(어차피 구현도 내가할듯)

5. 게임규칙 정하기 : 가장 먼저 진행해야 할 부분인것 같음. (컨텐츠같은것도 포함)

6. 기타 아이템 생성 : 딱히 중요하지는 않은듯 함.

우선순위는…..  5번 규칙정하기 -> 4번 클라이언트(솔로) - > 3번 서버 + 4번의 클라이언트(멀티) -> 6번 아이템생성 -> 다시 3,4,5번 재설정
	-> 2번 설치파일 만들기 -> 1번 홈페이지 개설 -> 운영
